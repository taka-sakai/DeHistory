Privacy Policy

Extension: DeHistory | Last updated: 2026-01-17

Overview

DeHistory is a browser extension that provides automatic and manual deletion of browsing history, whitelist, and automation features. This policy explains how data is handled in relation to the extension.

Information We Collect

  • Browsing Data: The extension uses the `browsingData` API to remove browsing history and related data. The extension does not collect, store, or transmit the contents of users' browsing history to external servers.
  • Local Settings: User preferences such as whitelists, auto-run settings, and UI options are stored locally using the `storage` API.
  • Tab and Window Info: The extension may access basic metadata (tab IDs, window IDs, active state) via the `tabs` and `windows` APIs to determine deletion targets and provide contextual UI. These are used only for deletion logic and UI purposes.
  • Personal Data: The current implementation does not collect personal identifying information (e.g., name, email) by default, nor does it transmit browsing content externally.

Permissions and How We Use Them

  • storage: To save and read user settings (whitelist, options, state) locally.
  • browsingData: To delete user-specified ranges of browsing history, cache, and related data.
  • tabs / windows: To identify targets for deletion, provide contextual UI, and perform explicit user-initiated operations.

Purpose of Use

The primary purpose is to perform deletion operations requested by the user. Secondarily, local data is used to retain user settings and improve usability. The current version does not send usage data or browsing data to external servers. If telemetry or remote error reporting is introduced in the future, it will be disclosed and require user consent.

Data Retention

Local settings are retained until the user deletes them or uninstalls the extension. Deletion via the `browsingData` API is executed immediately by the browser; the extension does not retain the content of deleted browsing data.

Third-Party Sharing

We do not sell or share users' browsing history or personal information with third parties. Disclosure will only occur when required by law or to protect rights, subject to applicable legal processes.

Security

Access to local storage is limited to the minimum necessary. The extension is designed not to transmit browsing data externally. Absolute security cannot be guaranteed—vulnerabilities in the browser or extension environment may pose risks.

User Rights and Choices

Users can review and delete extension settings via the extension's options page or by uninstalling the extension. Permissions can be revoked or the extension uninstalled from the browser's extension management UI.

Children

The extension is not specifically designed for children. Parental supervision is recommended for minors.

Changes to This Policy

We will publish updates to this policy and indicate the effective date. Significant changes will be announced via the extension description or UI when feasible.
